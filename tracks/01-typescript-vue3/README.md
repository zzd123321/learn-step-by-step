# TypeScript / Vue 3 学习路线

## 阶段目标

从已有 Vue 2、Element UI、Webpack 和接口联调经验出发，逐步迁移到 Vue 3、Composition API、TypeScript 类型系统和现代前端工程化。

## 预计模块

1. TypeScript 基础类型与类型推导
2. 函数、对象、数组、联合类型与泛型
3. Vue 3 Composition API 基础
4. Vue 3 组件通信与状态组织
5. Vue 3 + TypeScript 表单、接口和异步数据流
6. Vite、项目结构与工程化规范
7. 从 Vue 2 思维迁移到 Vue 3 实战

## 目录说明

- `lessons/`：课程正文，每节课只推进一个知识点或小模块。
- `examples/`：课程配套可运行代码。
- `README.md`：本路线目标、模块和目录说明。
- `PROGRESS.md`：本路线学习进度记录。
