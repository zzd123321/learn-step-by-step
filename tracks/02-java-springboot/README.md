# Java / Spring Boot 后端开发学习路线

## 路线定位

本路线面向已有前端开发经验、但 Java 后端经验较少的学习者。学习方式以“Java 与 JavaScript 对比 + 可运行示例 + 真实接口场景”为主，从 Java 基础语法逐步过渡到 Spring Boot 后端工程开发。

## 学习目标

完成本路线后，应能：

- 理解 Java 程序从源码到 JVM 运行的完整流程。
- 掌握 Java 基础语法、面向对象、集合、异常、泛型、函数式编程和常用标准库。
- 能使用 Maven 或 Gradle 管理后端项目依赖、测试、日志和配置。
- 能理解 JDBC、连接池、事务、MyBatis 或 JPA 的基本工作方式。
- 能使用 Spring Boot 编写分层清晰的 RESTful API。
- 能从前端接口联调视角理解参数校验、统一异常、认证授权、缓存和消息队列等后端能力。
- 能完成一个小型 Java 后端项目，并与接口文档和前端调用流程协作。

## 课程路线

### 阶段 1：Java 运行模型与基础语法

1. JDK、JRE、JVM、`.java` 到 `.class` 的编译执行流程、项目结构
2. 基础语法、变量、基本类型、引用类型、控制流、方法、数组
3. Java 与 JavaScript 在类型系统、运行时、作用域和错误暴露方式上的差异

### 阶段 2：面向对象与核心语言机制

1. 类、对象、构造方法、封装
2. 继承、多态、方法重写、访问修饰符
3. 接口、抽象类、组合优先于继承
4. `String`、包装类型、异常体系、泛型

### 阶段 3：集合、函数式编程与常用标准库

1. 集合框架：`List`、`Set`、`Map`、队列及其使用边界
2. Lambda 表达式、函数式接口、Stream API、Optional
3. 文件 I/O、NIO、日期时间 API
4. 注解、反射基础及其在框架中的作用

### 阶段 4：并发与 JVM 基础

1. 线程、线程生命周期、共享状态问题
2. 线程池、锁、并发容器
3. `CompletableFuture` 与异步任务编排
4. JVM 内存区域、垃圾回收基础、常见性能排查入口

### 阶段 5：工程化基础

1. Maven 或 Gradle 项目结构、依赖管理、生命周期
2. 单元测试、断言、测试边界
3. 日志、配置管理、多环境配置
4. 后端项目中的包结构与分层习惯

### 阶段 6：数据库访问

1. JDBC、连接池、SQL 注入防护
2. 事务、隔离级别、回滚场景
3. MyBatis 或 JPA 的基本使用
4. Repository / Mapper 与业务层的职责边界

### 阶段 7：Spring Boot Web 开发

1. Spring Boot 项目结构与启动流程
2. IOC（控制反转）与 DI（依赖注入）
3. Controller、Service、Repository 分层
4. RESTful API、请求参数、响应结构
5. 参数校验、统一异常处理、接口错误码
6. 认证授权基础

### 阶段 8：后端进阶与项目实战

1. 缓存、Redis 的典型使用场景
2. 消息队列与异步解耦
3. 基础分布式概念：幂等、限流、超时、重试
4. 接口文档协作、联调流程、项目实战

## 每节课产物

每节课只推进一个知识点或一个紧密相关的小模块，并创建：

- `lessons/Lxxx-英文短标题.md`：课程正文
- `examples/Lxxx-英文短标题/`：最小可运行代码

课程正文采用类似官方学习文档的写法，不再使用冗长固定模板。每节课围绕一个知识点展开，重点保证：

- 定义、适用场景、关键规则和边界条件清楚。
- 示例代码小而完整，可以真实编译运行。
- 关键语句、执行流程、参数含义、常见误区和真实项目用法讲清楚。
- 需要时补充运行方式、验证结果、练习和下一步建议。

如果课程中标记“已验证”，必须真实执行过对应命令；如果未执行或失败，必须如实说明。
