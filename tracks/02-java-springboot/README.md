# Java / Spring Boot 学习路线

## 阶段目标

从 Java 基础开始，逐步建立后端开发所需的语言能力、面向对象思维、Spring Boot Web 开发能力、接口设计能力和分层架构意识。

## 预计模块

1. Java 开发环境与基础语法
2. 类、对象、接口、集合与异常
3. Maven 与后端项目结构
4. Spring Boot 入门与 REST API
5. Controller、Service、Repository 分层
6. 参数校验、错误处理与日志
7. 数据库接入、事务与接口联调

## 目录说明

- `lessons/`：课程正文，每节课只推进一个知识点或小模块。
- `examples/`：课程配套可运行代码。
- `README.md`：本路线目标、模块和目录说明。
- `PROGRESS.md`：本路线学习进度记录。
