# Python / FastAPI 学习路线

## 阶段目标

从 Python 基础开始，逐步掌握脚本能力、虚拟环境、包管理、FastAPI 接口开发、异步处理和轻量后端服务搭建。

## 预计模块

1. Python 环境、语法与数据结构
2. 函数、模块、异常与文件读写
3. 虚拟环境、依赖管理与项目结构
4. FastAPI 基础路由与请求响应
5. Pydantic 数据模型与参数校验
6. 异步接口、流式响应与外部 API 调用
7. Python 服务测试、配置与部署基础

## 目录说明

- `lessons/`：课程正文，每节课只推进一个知识点或小模块。
- `examples/`：课程配套可运行代码。
- `README.md`：本路线目标、模块和目录说明。
- `PROGRESS.md`：本路线学习进度记录。
