# 数据库 / Redis / 网络 / 服务端基础设施学习路线

## 路线目标

本路线面向已经具备 Vue 前端和接口联调经验的学习者，目标是从后端视角建立数据存储、缓存、网络协议、接口设计、安全边界和基础设施部署能力。

完成本路线后，你应该能回答并实践这些问题：

- 一次浏览器请求从 DNS、TCP、TLS、HTTP 到服务端代码、数据库写入之间经历了什么。
- 如何设计关系型数据库表结构、主键、外键、约束、索引和事务边界。
- 如何阅读 SQL、执行计划和慢查询现象，并给出可落地的优化方向。
- Redis 适合解决什么问题，不适合解决什么问题，以及缓存异常如何预防。
- 如何设计可维护的 REST API，包括状态码、幂等性、限流、重试和超时。
- Cookie、Session、JWT、CSRF、XSS、CORS 的边界分别在哪里。
- Nginx、Docker、环境变量、日志和部署配置如何参与一个真实后端系统。

## 学习原则

- 每节课只推进一个可消化的小模块。
- 所有学习内容使用中文，英文术语必须附带中文解释。
- 每节课都包含可验证材料，例如 SQL、HTTP 请求样例、Docker Compose 配置或最小代码项目。
- 不启动高风险服务，不删除已有数据库，不提交数据库数据文件。
- 使用 Docker 前先检查 Docker 是否可用；没有本地数据库时优先使用纯 SQL、SQLite 或模拟数据。

## 阶段路线

### 阶段 1：请求链路与关系型数据库入门

1. L001：浏览器一次请求到服务端数据库之间到底发生了什么
2. L002：关系型数据库、表、行、列、主键与外键
3. L003：DDL：创建表、修改表、约束与默认值
4. L004：DML：插入、更新、删除与查询的基本安全边界
5. L005：范式与反范式：为什么表不能随便拆，也不能无限合并

### 阶段 2：SQL 查询能力

1. 条件查询、排序、限制返回数量
2. 聚合函数、分组统计与 HAVING
3. INNER JOIN、LEFT JOIN 与数据缺失分析
4. 子查询与派生表
5. 窗口函数：排名、分区统计与前后行比较
6. 分页查询：OFFSET 的问题与游标分页

### 阶段 3：索引、执行计划与性能

1. B-Tree 索引的心智模型
2. 单列索引、联合索引与最左前缀
3. 覆盖索引、回表与排序优化
4. 执行计划：如何看懂扫描方式、行数估算和过滤条件
5. 慢查询定位流程
6. 分页、模糊查询和热点数据的优化策略

### 阶段 4：事务、锁与并发控制

1. 事务的 ACID：原子性、一致性、隔离性、持久性
2. 隔离级别：读未提交、读已提交、可重复读、串行化
3. 脏读、不可重复读、幻读
4. MVCC：多版本并发控制的直觉
5. 行锁、间隙锁、表锁与锁等待
6. 死锁现象、分析与规避

### 阶段 5：MySQL 或 PostgreSQL 实际开发使用

1. 本地开发数据库连接配置
2. 建表迁移与版本化管理
3. 数据初始化与测试数据
4. 连接池、超时和最大连接数
5. 备份、恢复与只读账号
6. 常见生产问题：连接耗尽、慢查询、磁盘空间不足

### 阶段 6：Redis 与缓存工程

1. Redis 的定位：内存数据库、缓存和轻量队列
2. String、Hash、List、Set、Sorted Set 的使用场景
3. 过期时间、淘汰策略与内存风险
4. 缓存旁路、读写穿透与一致性取舍
5. 缓存穿透、缓存击穿、缓存雪崩
6. 分布式锁的边界和误用风险

### 阶段 7：网络协议与接口链路

1. DNS：域名如何找到服务器
2. TCP：连接、重传、拥塞控制的基本直觉
3. HTTPS 与 TLS：证书、握手和加密
4. HTTP 请求、响应、头部、状态码与缓存
5. WebSocket：双向长连接
6. SSE：服务端事件推送
7. 代理、网关和链路追踪

### 阶段 8：REST API 设计与可靠性

1. 资源建模与 URL 设计
2. GET、POST、PUT、PATCH、DELETE 的语义
3. 状态码与错误响应结构
4. 幂等性：重复请求为什么不能重复扣款
5. 限流、重试、退避和超时
6. API 版本、兼容性和弃用策略

### 阶段 9：浏览器安全与身份认证

1. CORS：跨域资源共享的真实边界
2. Cookie、Session 与服务端会话
3. JWT：无状态令牌与撤销难题
4. CSRF：跨站请求伪造
5. XSS：跨站脚本攻击
6. 登录态、刷新令牌和权限校验

### 阶段 10：Nginx、部署与基础设施

1. Nginx 反向代理
2. 静态资源、API 转发和路径重写
3. 负载均衡与健康检查
4. 访问日志、错误日志与排查流程
5. Docker 镜像、容器、网络和卷
6. Docker Compose 编排本地开发环境
7. 环境变量、配置文件和密钥管理

### 阶段 11：完整链路实践

1. 从 Vue 发起请求到后端 Controller
2. 参数校验、鉴权与业务服务
3. 数据库事务落库
4. Redis 缓存读取与失效
5. Nginx 反向代理与日志定位
6. Docker Compose 一键启动开发环境
7. 常见故障演练：超时、跨域、锁等待、缓存不一致

## 目录说明

- `lessons/`：课程正文，每节课只推进一个知识点或小模块。
- `examples/`：课程配套可运行材料，包括 SQL、HTTP 请求样例、配置或最小程序。
- `README.md`：本路线目标、模块和目录说明。
- `PROGRESS.md`：本路线学习进度记录。
