# 数据库 / 网络学习路线

## 阶段目标

系统建立全栈开发所需的数据库、SQL、事务、索引、HTTP、网络排查和基础工程化能力，服务于后端开发、接口联调和项目部署。

## 预计模块

1. 关系型数据库与 SQL 基础
2. 表设计、主键、外键与约束
3. 查询、聚合、连接与分页
4. 索引、事务与常见性能问题
5. HTTP 请求响应、状态码与缓存
6. DNS、TCP、TLS 与常见网络排查
7. 配置、日志、部署与工程化基础

## 目录说明

- `lessons/`：课程正文，每节课只推进一个知识点或小模块。
- `examples/`：课程配套可运行代码。
- `README.md`：本路线目标、模块和目录说明。
- `PROGRESS.md`：本路线学习进度记录。
