# AI 应用开发学习路线

## 阶段目标

在前后端、接口、数据和工程化基础之上，学习大模型 API 调用、Prompt 设计、流式输出、RAG、工具调用和 AI 产品工程化。

## 预计模块

1. 大模型 API 基础调用
2. Prompt 结构化设计与输出约束
3. 流式输出与前端实时展示
4. Embedding、向量检索与 RAG 入门
5. 工具调用与业务系统集成
6. AI 应用的错误处理、评估与日志
7. 从 Demo 到可维护产品的工程化实践

## 目录说明

- `lessons/`：课程正文，每节课只推进一个知识点或小模块。
- `examples/`：课程配套可运行代码。
- `README.md`：本路线目标、模块和目录说明。
- `PROGRESS.md`：本路线学习进度记录。
